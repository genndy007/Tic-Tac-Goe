Tic-Tac-Goe
---------------
It's a simple implementation of a classic game for two, Tic-Tac-Toe, but for this time in Golang.

How to run
------------
Way 1:     
```go run .```    
Way 2:     
```go build```      
```./tic-tac-goe```    

Rules
-----------
Just type number of cell you want to place your player, X or O.
